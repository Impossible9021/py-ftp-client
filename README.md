# py-ftp-client  
A tiny FTP/FTPS client written in Python 3 for Linux  
Uses only standard libs so it can be easily run on any system.  
License: MIT.  
Thanks to cs.brown.edu & dlptest.com for free FTP servers.  
  Features:  
  + FTP / FTP+SSL support  
  + Bash-like CLI  
  + Anon login support
